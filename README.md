## 你是不是也這樣？

- 📰 打開財經新聞，**每個字都認識，連在一起就看不懂**
- 😰 大盤跌了，想知道「今天該緊張嗎」，查到的卻是一堆術語
- 🤷 買了股票之後，其實**不知道它現在是貴還是便宜**

晨鐘就是為這三件事而生的。

## 它幫你做四件事

### 📰 今日晨報
每天早上打開，四段話講完：昨晚美股發生什麼、市場現在的氣氛、你關注的股票有沒有事、接下來有什麼大日子。捲完就結束，不給你資訊焦慮。

### 🚦 市場晨燈
把複雜的市場狀態濃縮成一盞燈——
**綠燈**：市場平穩。**黃燈**：訊號分歧。**紅燈**：市場緊張。
每盞燈都附一句人話理由，忠實描述現況，不給任何行動指示。判斷規則全部公開在 App 的「運作原理」頁。

### 🩺 個股健檢
輸入股票代號，三張卡告訴你：
**貴不貴**（本益比翻譯成「要幾年才回本」）、**體質好不好**（五項體檢，像健康檢查報告）、**有沒有紅旗**（虧損、高負債、燒錢……直接列給你看）。

### ⭐ 觀察清單
把你關心的股票加進清單，之後每天的晨報自動幫你盯著——大漲大跌、接近新高新低、什麼時候公布財報，都會主動告訴你。

## 🚀 在自己電腦上跑起來

**Windows 三步驟（不用會程式）：**

1. 安裝 [Python](https://www.python.org/downloads/)（安裝時勾選 "Add Python to PATH"）
2. 點右上角綠色 **Code → Download ZIP**，下載後解壓縮
3. 雙擊資料夾裡的 **`start.bat`** —— 完成！瀏覽器會自動打開晨報

之後每天要看，就雙擊 `start.bat`。

**Mac / Linux：**

```bash
git clone https://github.com/bargisula/morningbell.git
cd morningbell
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --port 8888
# 打開 http://127.0.0.1:8888
```

## ✨ 選配：AI 主持人導讀

預設的晨報是模板文案，已經夠用。如果你想要一位「AI 電台主持人」每天用聊天的口吻幫你導讀行情：

1. 到 [console.anthropic.com](https://console.anthropic.com) 申請一把 API 金鑰（有免費額度）
2. 把 `.env.example` 複製成 `.env`，填入 `ANTHROPIC_API_KEY=你的金鑰`
3. 重新啟動，晨報裡就會多一段 AI 導讀

不設定也完全不影響其他功能。

## ❓ 常見問題

**要收費嗎？**
不用。晨鐘是開源軟體，跑在你自己的電腦上。資料來自 Yahoo Finance 的免費公開資料（延遲報價）。

**我的觀察清單會被上傳嗎？**
不會。所有資料都存在你電腦上的一個小檔案裡（`data/` 資料夾），不會離開你的電腦。

**它會叫我買什麼股票嗎?**
不會，永遠不會。晨鐘只把公開數據翻譯成人話，買賣永遠是你自己的決定。

**資料多即時？**
Yahoo Finance 的免費資料有 15 分鐘左右的延遲。對「每天看一次晨報」的用途來說綽綽有餘；如果你需要秒級報價，這不是為你設計的工具。
