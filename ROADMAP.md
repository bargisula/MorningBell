# 晨鐘 MorningBell 開發路線圖

> 每一版最多 3 個主要新功能——我們寧可小而精，也不做大而雜。
> 核心承諾不變：零金鑰可用、忠實描述不給建議、資料不離開你的電腦。

## ✅ V1 — 看懂今天（已發布）

- 📰 今日晨報：四段式，捲完就結束
- 🚦 市場晨燈：綠 / 黃 / 紅 + 一句忠實描述的理由
- 🩺 個股健檢:貴不貴、體質好不好、有沒有紅旗
- ⭐ 觀察清單：晨報自動追蹤
- 🔍 運作原理：所有判斷規則公開透明

## 🚧 V2 — 每天回來的理由

主題:晨報講「昨天」，V2 補上「明天」與「過去」。

| 功能 | 說明 |
|------|------|
| 📅 **本週大事日曆（主打）** | FOMC 利率決議、CPI、非農就業、財報季——「這週四有大事」一目了然。不只你的清單，整個市場的大日子都幫你標好 |
| 🌡️ 板塊溫度計 | 11 個產業當日資金流向色塊圖，「今天錢往哪裡跑」一眼看懂 |
| 🕯️ 晨燈歷史帶 | 過去 90 天的燈號色帶 + 每日晨報存檔，「上次紅燈是哪天」可回溯、可驗證 |

小改進：財經名詞即點即解、手機版排版優化。

## 🔭 V3 — 從看懂到陪伴

主題：工具開始認識你。

| 功能 | 說明 |
|------|------|
| 💬 「這是什麼意思？」 | 晨報與健檢的每個數字旁都能追問，AI 基於當下數據用白話回答（自備金鑰選配，與 V1 的 AI 導讀相同模式） |
| 📈 健檢 2.0 | 不再用通用門檻——跟「這家公司自己的五年區間」和「同業中位數」比貴俗 |
| 📬 晨報送上門 | 每天早上自動推送晨報（管道規劃中） |
| 🗓️ 月度回顧 | 「你的清單這個月發生了什麼」，基於 V2 的晨報存檔自動生成 |

## ❌ 永遠不做的事

- 選股推薦、目標價、買賣訊號
- 任何形式的市場預測
- 收集使用者資料
